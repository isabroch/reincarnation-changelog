Pathbuilder JSONs

For each sheet, go to Menu > Export JSON > View JSON. Copy all the text there into the respective text areas. Make sure you copy one JSON in before doing the next. Pathbuilder only stores one json per user, and overrides the old one when you hit Export JSON.

This tool simply checks differences between sheets. You'll still need to double check the legality of your options - i.e. if you lost access to feats, did all your skill/ability increases, and so on.

Unfortunately, when special feature changes (any features with a special menu of its own, i.e. kineticist gate junctions, champion causes, etc.) happen aren't tracked well, so you will need to manually add them to your log at the levels that those changes happened. There is a summary of any special changes to your build at the end out the output for reference.
